Versão 1 — Split: tabela editável + timeline

Recomendada para o Hub

Esquerda: etapas, atividades e subtarefas com colunas editáveis. Direita: barras sincronizadas. É o que mais se aproxima do que você pediu (concluir etapa, sub-etapas, edição inline).

  • Implementação um pouco mais complexa

Versão 3 — Swimlanes por fase (Kanban + tempo)

Visual / gestão ágil

Quatro colunas = Planejamento, Produção, Revisão, Entrega. Cada atividade é um card com mini-barra de progresso. Menos “Gantt puro”, mais visão de fluxo por etapa.
