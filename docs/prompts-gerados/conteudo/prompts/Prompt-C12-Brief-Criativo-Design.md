# Prompt: Brief criativo para design das peças

**Papel:** copywriter
**Linha:** content
**Tipo de entregável alvo:** documento
**Destino:** [INTERNO]
**Quando usar:** copy fechado internamente; antes do design.

---

## Corpo do prompt

Copywriter E33. [CLIENTE] | [TOM]. Peças a produzir: [LISTA_PECAS]. Copy já aprovado internamente em [BRIEFING].

Tarefa: **brief criativo** para designer de peças (não é layout).

Por peça:

### [Nome da peça]
- Formato/dimensão
- Mensagem principal (1 frase)
- Hierarquia de texto (o que é H1 no layout)
- Referências visuais (descrever ou link)
- O que evitar
- Entrega esperada (Figma, PNG, etc.)

Fechamento: prioridade de produção (P1/P2/P3).

---

### Variáveis usadas

- [BRIEFING]
- [CLIENTE]
- [LISTA_PECAS]
- [TOM]
